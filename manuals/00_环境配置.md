1、使用conda创建新环境

```bash
#创建新环境
conda create -n nnunet python=3.10 -y
#查看电脑安装的CUDA版本，根据CUDA版本安装对应的pytorch
nvidia-smi

conda activate nnunet

#conda总是网络失败，因此用pip安装pytorch
# 1. 先配置清华镜像（你已经能访问这个页面，说明网络没问题）
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/conda-forge/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/pytorch/

# 2. 设置搜索时显示通道地址，方便确认是否从清华源下载
conda config --set show_channel_urls yes

# 3. 清除之前的索引缓存（重要！）
conda clean --all -y

# 4. 安装 PyTorch (CUDA 11.8 版本)


#检查pytorch安装情况
python -c "import torch; print(torch.__version__); print(torch.cuda.is_available())"
```

注意：使用pip install -e .安装的torch是CPU版本，如果需要GPU版本的需要先自己安装。
